<b1>Proyekt 2 hissədən ibarət olmalıdır. Admin panel və User panel (İstəyə görə 2
paneldən birinə login olunacaq).
1) Admin panel hissəsi:
Admin restoranı idarə edəcək.
Menu – ya yeni yeməklər əlavə edəcək, yeməklərə hansı inqrediyentlər
gedəcəyini yə onların sayını təyin edəcək. (Hər bir məhsulun mütləq inqrediyenti
olmalıdır). Hər bir yemək haqqda əlavə informasiya daxil olunmalıdır.
Restoranda mövcud olan inqrediyentlər skladda saxlanılacaq. Hər bir zakazdan
sonra məhsulda olan inqrediyentlər skladda olan inqrediyentlərdən çıxılacaq.
Admin sklada yeni inqrediyentlər əlavə edə bilər və ya onların sayını artıra bilər.
Hər bir artırılmış yeni inqrediyentə görə restoranın büdcəsinnən pul çıxılacaq (Hər
bir inqrediyentin kilosuna görə qiyməti olacaq).
Admin restoranın vəzyətini incələmə qabiliyyətinə malik olmalıdır (Statistika).
Aylıq (Günlük də bura daxildir) qazanc, xərc.
 <hr>
2) User panel hissəsi:
Hər bir user daxil olmadan əvvəl register olmalıdır.
User daxil olduqdan sonra yeməkləri zakaz etməyə başlaya bilər.
Hər bir yemək səbətə əlavə olunmalıdır. User səbətə keçid etdikdən sonra
müəyyən dəyişikliklər edə bilməlidir səbətdə (Yeməkləri səbətdən çıxarmaq,
yeməklərin içindəki inqrediyentləri dəyişmək, Məsələn: pizza extra pendirli
olmalıdır. Bu zaman normalda pizzaya 50 qr pendir gedirdisə bu zaman 100 qr
gedəcək). Sonda isə zakazı təsdiqliyərək ödənişə keçid edir.
User yeməklər haqqda informasiyaya da baxa bilər (Məsələn info button ola
bilər).
User ödəniş etdikdən sonra ödənilmiş məbləğ restoranın büdcəsinə əlavə
olunmalıdır. Yeməklər üçün istifadə olunan inqrediyentlər isə skladdan çıxılmalıdır.
Əgər müəyyən yemək üçün inqrediyent çatmırsa o menu – dan müvəqqəti
qaldırılmalıdır və yalnız sklada inqrediyent əlavə olunan zaman menu – ya yenidən
gəlməlidir AVTOMATİK.
User history - ə sahibdir. Oraya daxil olub etdiyi zakazları və onların tarixini
incələmək və s.
3) Əlavə qydlər:
Proyektin hər bir gedişatı fayla yazılmalıdır. Userlər, yeməklər, inqrediyentlər,
restoranın statistikası və s.
Hər bir exception catch olunmalıdır.
Mükəmməl dizayn olunmalıdır. (Sadə dizayn qəbul olunmur)...............
